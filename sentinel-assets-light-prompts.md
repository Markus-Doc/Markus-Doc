# Sentinel Assets Light — ImageGen Prompt Pack

This pack is structured for batch generation of `sentinel-assets-light.zip` using an image generation connector.

Global style for all prompts:
- pleasant light cybersecurity / mission-control interface aesthetic
- warm off-white background `#f7f9fb`, soft mist-blue panels `#eaf4ff`
- accents: cyan `#00aeea`, mint `#22c78a`, pale slate text `#253140`
- thin hairline grids, subtle frosted-glass overlays, faint paper grain
- sharp geometric forms, generous negative space
- no people faces, no real-company logos, no AI mascots/emojis
- cinematic, restrained, premium polish
